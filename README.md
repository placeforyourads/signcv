### Название проекта: SignCV
### ОПИСАНИЕ
Программа для распознавания дактиля русского жестового языка. 

Идея программы проста - популяризировать распространение электронных сервисов среди немых и облегчить коммуникацию между говорящими и немыми

Я занялся этой темой, потому что какое-то время уже изучал русский жестовый язык и хотел углубиться в своих познаниях

### Дополнительные библиотеки для установки
Для корректной работы должен быть установлен _Python 3.12_ и библиотеки _cv2, numpy, mediapipe, pillow_

### Запуск программы
Нужно запускать файл _main.py_. 

Примеры того, как должна работать программа, лежат в папке _recordings_and_testings_
